# Simple Cryptocurrency Wallet Simulator

This is a simple cryptocurrency wallet simulator written in Python. It demonstrates basic wallet functionality like generating addresses, adding funds, and sending transactions.

## Features
- Generate a wallet with a unique address.
- Add funds to a wallet.
- Send funds between wallets.
- Check wallet balance and details.

## How to Run
1. Clone the repository.
2. Run the `wallet.py` script:
   ```bash
   python wallet.py
